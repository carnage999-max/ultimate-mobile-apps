# App Icons

Place your app icons in this directory with the following filenames:

- `vita.png` - VitaChoice® icon
- `timer.png` - Timer for Life icon
- `freedom.png` - FreedomTek® icon
- `liberty.png` - LibertySocial™ icon
- `legal.png` - LegalTraker™ icon

**Recommended specifications:**
- Format: PNG with transparency
- Size: 512x512 pixels (or higher, will be optimized by Next.js)
- Aspect ratio: 1:1 (square)

The icons will be automatically optimized by Next.js Image component.
